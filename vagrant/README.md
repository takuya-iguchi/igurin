## boxの設定
$ cd d: <br>
$ cd d:\vagrant <br>
$ vagrant box add {title} {url} <br>
$ vagrant init {title} <br>
$ vagrant up <br>
$ vagrant halt <br>

http://www.vagrantbox.es/
